### Project Description: Personal Portfolio Website

A fully responsive and modern personal portfolio website built to showcase my skills, projects, and professional experience. The front end is developed with a focus on clean design, smooth user interaction, and optimal performance across all devices.

**Live Demo:** [https://zainab-fatima.netlify.app/](https://zainab-fatima.netlify.app/)

#### Technologies & Frameworks Used

*   **HTML5:** Semantic markup for structure, accessibility, and SEO.
*   **CSS3:** Custom styles for typography, layout, and unique design elements, including CSS Grid and Flexbox.
*   **JavaScript (ES6+):** Implemented for dynamic functionality, such as interactive elements and enhancing the user experience.
*   **Bootstrap 5:** Utilized primarily for the responsive grid system and pre-built components to ensure mobile-first responsiveness and accelerate development.

#### Key Features

*   **Fully Responsive Layout:** The website seamlessly adapts to desktops, tablets, and mobile phones using a combination of Bootstrap's grid and custom CSS media queries.
*   **Modern UI/UX:** Features a clean, professional aesthetic with intuitive navigation.
*   **Interactive Elements:** Enhanced with JavaScript for a dynamic user experience.
*   **Performance Optimized:** Built with fast loading times and a smooth user interface in mind.
